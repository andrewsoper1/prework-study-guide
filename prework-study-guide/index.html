HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The title element defintes the title for a webpage
  • The style element defines the CSS styles associated with a webpage
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport
  • The script element embeds client-side scripts in the HTML document
  • The charset element sets our character encoding
  • The http-equiv element sets an HTTP header for content
  • The viewport element sets the default view for our webpage
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents with 1 being the largest
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items
  • The img element contains information about images that are displayed on the webpage
  • The src attribute within the img element is most important since it defines the location of the image
  • The br element creates a line of empty space between 2 blocks of content
  • The a element creates links to the same webpage or other webpages

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the contents inside an element
  • Declarations contain the CSS property we want to apply and the value of the property
  • Ex. declaration color: blue; includes the CSS property color and the value for that property, blue
  • The * selector can be used to precede properties to declare all elements on the page will have the same properties
  • By combining 2 selectors with a comma, you can allow declarations to affect both selectors
  • The img selector assigns property values to any image element in the markup
  • Make sure to link CSS files to HTML files for the changes to be applied

Git

  • git checkout brand name: switches to designated branch
  • git checkout -b branch name: creates a new branch and switches to it
